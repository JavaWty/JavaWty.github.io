git

发表于 2020-03-24 更新于 2020-12-21 分类于 工具 阅读次数：

基础

pwd 查看当前目录

git init 把这个目录变成Git可以管理的仓库

ls -a 查看当前目录下面文件 -a包含隐藏文件

git add <file> 告诉Git，把文件添加到暂存区：git add file2.txt file3.txt

git commit -m " " 告诉Git，把文件提交到仓库,并添加注释

本地操作

git status 实时掌握当前仓库的状态

git diff 查看具体的修改内容

git log 显示从最近到最远的提交日志

git log --pretty=oneline 以简洁的内容展示日志文件

cat <file> 查看文件内容

git reset --hard HEAD^ 查看上一个版本,HEAD^上一个, HEAD^^上两个, HEAD~100上100个

git reset --hard 版本号 根据版本号来确定回退到某一个版本

git reflog 用来记录你的每一次命令：

git diff HEAD -- (空格)<file> 查看工作区和版本库里面最新版本的区别

git checkout -- (空格)<file> 把某文件在工作区的修改全部撤销

rm <file> 删除本地的文件

git rem <file> 删除版本库中的文件

分支

git switch -c 分支名字 创建分支

git switch 分支名字 切换分支

git branch -d 分支名字 删除分支

git branch 查看当前分支

git merge A 合并A分支到当前分支

git merge --no-ff -m "描述信息" A 普通模式的合并,可以在log历史中查看到

git log --graph 查看分支合并图

远程

git remote add origin 地址 连接远程

git push -u -f origin master 首次推送本地文件到远程添加 -u,强制上传使用-f,以后直接 git push origin master

git clone url地址 克隆到本地
