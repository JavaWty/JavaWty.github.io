九大隐含对象

隐含对象：是我们在页面直接可以使用的对象

  1. PageContext pageContext=null； 当前对象
  2. HttpSession session=null； 会话对象
  3. Throwable exception=null； 当前捕获的异常对象
  4. ServletContext application=null； 整个web应用
  5. ServletConfig config=nul1； servlet配置信息
  6. JspWriter out=null； 输出数据的out对象
  7. object page=this； 当前jsp
  8. HttpServletRequest request， 本次请求的详细信息
  9. HttpServletResponse response 本次响应的详细信息

五大常规对象

  1. Throwable exception=null； 当前捕获的异常对象
  2. ServletConfig config=nul1； servlet配置信息
  3. JspWriter out=null； 输出数据的out对象
  4. object page=this； 当前jsp
  5. HttpServletResponse response 本次响应的详细信息

四大域对象

排列顺序是从小到大

PageContext pageContext=null； 当前对象

1)可以调用其他的八个隐藏对象,通过pageContext.getXXX();

2)只可以在当前页面共享数据,pageContext.setAttribute();

HttpServletRequest request， 本次请求的详细信息

1)同一次请求中共享数据

HttpSession session=null； 会话对象

1)同一次会话共享数据,开启关闭浏览器称为一次会话

ServletContext application=null； 整个web应用

1)只要web应用不卸载,都可以在整个web中共享数据

域对象常用方法

  1. setAttribute()
  2. getAttribute()
  3. removeAttribute()
